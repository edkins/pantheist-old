Pantheist v0.0 - Documentation

Pantheist server

Static resources

/

Root page, alias for /static/index.html

/static

Static assets. These are not generated dynamically and requesting them should have no side-effects.

/static/doc

System documentation.

API

/syntax GET

Lists syntax structures 
{
    "resources":[
        {
            "path":"/syntax/mysyntax",
            "name":"My Syntax"
        }
    ]
}

/syntax/name GET PUT DELETE

Accesses a particular syntax structure.

Currently the PUT api only accepts an empty json object, creating a new empty resource. It will fail if there is already a resource with that name.
